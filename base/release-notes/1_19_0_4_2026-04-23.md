Hello everyone! We're pushing a small hotfix update today to tackle some outstanding issues with the 1.19 "Scribe" Update that was released on Monday. Check the changelog below for more information.

## Update 1.19.0.4 Changelog

-    Fixed an issue causing out-of-sync errors in cross-platform multiplayer.


-    Fixed an issue causing certain trait-related events (such as Lunatic and Depressed) to fire too frequently.


-    Resolved an issue with some High Medieval Attire clothing not displaying correctly when using the color picker.


-    Fixed a typo in the Sword of Japan Steam achievement description.


-    Fixed an issue where UI elements could go out of bounds in the Story Cycle panel.


-    The Court Visuals UI panel is now fixed in place and can no longer be moved.


-    Minor French localization fixes.
